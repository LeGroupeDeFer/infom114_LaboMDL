# infom114_LaboMDL

## Quickstart

1. install rust(cargo)

2. go nighty

3. setup mariadb on your machine (look at logins in .env & rocket.toml)

4. `cargo build` && `cargo run`

5. go on http://localhost:8000 


## Dockerization

1. Install docker and docker-compose

2. docker-compose up

3. go on http://localhost:8000
